# Open-the-folder-and-find-the-image-file

These scripts are scripts that open folders and put image files into rawimage objects.
이 스크립트들은 폴더를 열고, 이미지 파일을 rawimage 오브젝트에 넣는 스크립트입니다.
외부 api를 사용하지 않은 것이 특징입니다.
